# To_Do_List_Project

This is a simple console-based To-Do List project written in **C++**.  
It allows users to manage tasks using arrays, strings, and basic algorithms.

## Features
- Add tasks
- Show all tasks
- Mark tasks as done
- Delete tasks
- Search tasks
- Sort tasks alphabetically

## How to Run

### On Linux / Mac:
g++ main.cpp -o todo  
./todo

### On Windows:
g++ main.cpp -o todo.exe  
todo.exe
